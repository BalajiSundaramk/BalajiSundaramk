## Featured Project

**[TaskFlow](https://github.com/BalajiSundaramk/TaskFlow)** — natural-language task capture app
[Live](https://taskflow-mn89.onrender.com) · [Source](https://github.com/BalajiSundaramk/TaskFlow)

Removes the friction of manual categorization — you type a task, note, or reminder in plain language, and an intent-detection layer classifies it, extracts tags, and parses dates with `chrono-node` to auto-schedule reminders.

- **Backend** — Node.js, Express, JWT + bcrypt auth, role-based admin panel
- **Frontend** — React (Vite), voice input via the Web Speech API, client-side search
- **Production fix** — Render's ephemeral filesystem was silently wiping the SQLite database on every redeploy. Diagnosed the cause and migrated the persistence layer to Turso (cloud SQLite) with zero downtime.
