### 🌟 Featured Project — TaskFlow

**A full-stack productivity app** built end-to-end, deployed and live.

🔗 **Live Demo:** [taskflow-mn89.onrender.com](https://taskflow-mn89.onrender.com)
📁 **Repo:** [github.com/BalajiSundaramk/TaskFlow](https://github.com/BalajiSundaramk)

- Built with **React (Vite)** on the frontend and **Node.js / Express** on the backend
- Secure auth using **JWT + bcrypt**
- **Voice input** via the Web Speech API, dark mode, browser notifications, and an admin panel
- Migrated the database mid-project from local **SQLite** to **Turso (cloud SQLite)** after discovering Render's ephemeral filesystem was wiping local data on redeploys — a real production debugging story, not just a tutorial clone

> This project is the one to lead with in interviews — it shows you can ship, debug live infra issues, and make architecture decisions under real constraints.
