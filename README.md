## Featured Project

### [TaskFlow](https://github.com/BalajiSundaramk/TaskFlow) — Natural-language task capture app

**Live:** [taskflow-mn89.onrender.com](https://taskflow-mn89.onrender.com) · **Code:** [github.com/BalajiSundaramk/TaskFlow](https://github.com/BalajiSundaramk/TaskFlow)

TaskFlow removes the friction of picking categories manually — you type a task, note, or reminder in plain language, and an intent-detection layer classifies it, extracts tags, and parses any dates with `chrono-node` to auto-schedule reminders.

- Backend: Node.js, Express, JWT + bcrypt authentication, role-based admin panel
- Frontend: React (Vite), voice input via the Web Speech API, client-side search and filtering
- **Mid-project fix:** discovered Render's ephemeral filesystem was silently wiping the SQLite database on every redeploy. Diagnosed the root cause and migrated the persistence layer to Turso (cloud SQLite) without downtime.

This project is the one I'd want to walk an interviewer through — most of what I actually learned building it happened after the first version already worked.
